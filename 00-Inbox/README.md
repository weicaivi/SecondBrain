# Inbox

Zero-friction capture zone. Dump anything here — process later.

**What lands here:**
- Quick URL drops with no time to think
- Voice-to-text dumps
- Copy-pastes from anywhere
- Anything Claude couldn't auto-categorize

**Processing rule:** Once a week, empty this folder.
- Worth a permanent note → move to `03-Knowledge/` or `04-Sources/`
- Just a reference → move to `04-Sources/`
- Outdated / irrelevant → delete without guilt
