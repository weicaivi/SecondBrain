# Sources

Atomic source notes — one file per article, video, post, or paper worth keeping.

Use the template: `_System/Templates/source-note.md`

**Naming:** `YYYY-MM-DD author title.md`
Example: `2026-03-17 karpathy scaling-laws.md`

**What belongs here:** anything you want to reference later but haven't yet processed into a knowledge note.
**What does NOT belong here:** casual daily reads that don't need a permanent reference.
