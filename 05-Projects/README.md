# Projects

Active work and personal projects. One folder per project.

**Suggested structure per project:**
```
05-Projects/my-project/
├── README.md        Goal, status, key decisions
├── Log.md           Running work log (append-only, newest first)
└── Resources.md     Links, references, related people
```

Archive completed projects to `06-Archive/` — don't delete, you'll want the context later.
